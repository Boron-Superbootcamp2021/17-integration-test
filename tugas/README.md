# Task Manager App

a simple web app to manage task

## Quick Start

make sure you have nodejs > 14 installed on your system
to build this app, simply install & run

```bash
npm i
npm run dev
```

## Build

for optimized build

```bash
npm run build
```

built files are on `www` directory.

```bash
#  run any static server 
http-server --port=8080 ./www
```
